# Blog Ap REST API

Backend server for Blog Ap.

To run the application

1. Clone this repo
2. Run `npm install`
3. Run `npm start`

## Built With

* Node.js
* Express.js
* PstgreSQL

## Environmental Variables

### Database URL

You must add your MongoDB url `PG_DATABASE_URL` in the `index.js`
